# Premium Subscription — Implementation Plan

## Overview
Реализация платной подписки DotaPulse Pro с оплатой через ЮKassa. Ежемесячная подписка 199₽/мес.

## Architecture

### 1. Premium Status Storage (Redis)
```
Key:    premium:{steamId}
Value:  { plan: 'pro', expiresAt: number, paymentId: string }
TTL:    expiresAt - now + 86400 (1 day buffer)
```
Используем существующий `lib/kv.ts` (Upstash Redis).

### 2. Auth Flow Changes
- При логине (`/api/auth/callback`): проверяем Redis на наличие активной подписки
- Добавляем в JWT: `premium: boolean`, `premiumExpiresAt: number`
- При каждом запросе `/api/auth/me`: проверяем `premiumExpiresAt > Date.now()`

### 3. Middleware Updates
- Добавить проверку premium статуса в `middleware.ts` для premium API routes
- Расширить matcher на `/api/premium/*`

## Implementation Details

### `lib/premium.ts`
```typescript
export interface PremiumStatus {
  plan: 'pro';
  expiresAt: number;
  paymentId: string;
}

export async function getPremiumStatus(steamId: string): Promise<PremiumStatus | null>
export async function setPremiumStatus(steamId: string, status: PremiumStatus): Promise<void>
export async function removePremiumStatus(steamId: string): Promise<void>
export function isPremiumValid(status: PremiumStatus | null): boolean
```

### `app/api/premium/checkout/route.ts`
1. Принимает `{ steamId }` от авторизованного пользователя
2. Создаёт платёж в ЮKassa API:
   ```typescript
   POST https://api.yookassa.ru/v3/payments
   {
     amount: { value: '199.00', currency: 'RUB' },
     confirmation: { type: 'redirect', return_url: `${baseUrl}/premium/success` },
     capture: true,
     description: 'DotaPulse Pro — ежемесячная подписка',
     metadata: { steamId }
   }
   ```
3. Возвращает `confirmation_url` для редиректа пользователя

### `app/api/premium/webhook/route.ts`
1. Верификация вебхука через `Idempotency-Key` header
2. При `succeeded` статусе: `setPremiumStatus(steamId, { plan: 'pro', expiresAt: now + 30 дней, paymentId })`
3. При `canceled`/`refunded`: `removePremiumStatus(steamId)`
