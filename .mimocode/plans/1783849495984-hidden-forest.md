# Profile Page — Animated CSS Background

## Goal
Replace the pure black background on the profile page with a beautiful CSS-only animated background — ambient light orbs, subtle gradient, and depth.

## Concept
Multiple layered radial gradients creating "ambient light orbs" that slowly animate:
- A deep dark base (`#08080f`)
- 3-4 soft colored orbs (dark red, dark blue, dark purple) positioned at edges
- Very slow CSS animation (20-30s cycle) so orbs drift gently
- Faint dot grid overlay for tech/gaming feel
- Everything subtle — doesn't compete with content

## File to modify
**`app/players/[id]/PlayerProfile.module.css`** — add background styles to `.profileContainer`

## Implementation

Add to `.profileContainer`:
```css
background: #08080f;
position: relative;
overflow: hidden;
```

Add `::before` pseudo-element for animated orbs:
```css
.profileContainer::before {
  content: '';
  position: fixed;
  inset: 0;
  background:
    radial-gradient(ellipse 600px 400px at 10% 20%, rgba(255, 76, 76, 0.07) 0%, transparent 70%),
    radial-gradient(ellipse 500px 500px at 85% 15%, rgba(80, 60, 180, 0.06) 0%, transparent 70%),
    radial-gradient(ellipse 400px 300px at 70% 80%, rgba(255, 119, 0, 0.04) 0%, transparent 70%),
    radial-gradient(ellipse 350px 350px at 20% 85%, rgba(60, 100, 200, 0.05) 0%, transparent 70%);
  animation: ambientDrift 25s ease-in-out infinite alternate;
  pointer-events: none;
  z-index: 0;
}
```

Add `::after` for dot grid:
```css
.profileContainer::after {
  content: '';
  position: fixed;
  inset: 0;
  background-image: radial-gradient(rgba(255,255,255,0.03) 1px, transparent 1px);
  background-size: 24px 24px;
  pointer-events: none;
  z-index: 0;
}
```

Add animation keyframes:
```css
@keyframes ambientDrift {
  0% { transform: translate(0, 0) scale(1); }
  33% { transform: translate(30px, -20px) scale(1.05); }
  66% { transform: translate(-20px, 15px) scale(0.95); }
  100% { transform: translate(10px, -10px) scale(1.02); }
}
```

Ensure all content sits above the background:
```css
.profileHero, .statsGrid, .profileClient, .privacyWarning, .errorContainer {
  position: relative;
  z-index: 1;
}
```

## Verification
1. Open `/players/{steamId}` — background has subtle animated color orbs
2. Orbs move very slowly (barely noticeable, adds life)
3. Dot grid is very faint — adds texture without distraction
4. Content sits cleanly on top
5. No performance issues (CSS-only, GPU-accelerated transforms)
