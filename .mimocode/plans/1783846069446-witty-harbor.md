# Plan: Улучшение шапки (Navbar) DotaPulse

## Текущее состояние

Файлы: `app/components/Navbar/Navbar.tsx` (114 строк), `app/components/Navbar/Navbar.module.css` (287 строк)

**Проблемы:**
1. **Дизайн:** статичный фон без привязки к скроллу, устаревший glassmorphism, подсветка активной страницы только подчёркиванием, нативный `<select>` для языка
2. **Мобильная версия:** два отдельных выпадающих меню (навигация + экшены), нет анимации открытия/закрытия, нет overlay, бургер не анимируется в крестик

## Фаза 1 — CSS-улучшения (десктоп)

**Файл: `app/components/Navbar/Navbar.module.css`**

### 1.1 Scroll-aware фон
- Добавить состояние `scrolled` через JS (скролл > 20px)
- В состоянии `.scrolled`: плотнее glass (`blur(16px) saturate(1.2)`), уменьшенный padding (`10px 60px`), более непрозрачный фон

### 1.2 Улучшенный glassmorphism
- Заменить `rgba(26,26,26,0.85)` на `rgba(20,20,22,0.72)` с `saturate(1.2)`
- Добавить тонкую градиентную полосу снизу (от `rgba(255,76,76,0.3)` к прозрачному)

### 1.3 Индикатор активной страницы
- Заменить underline на pill-индикатор: `background: rgba(255,76,76,0.12); border-radius: 6px; padding: 4px 10px`

### 1.4 Язык-переключатель (toggle вместо select)
- Две кнопки `RU | EN` в pill-контейнере вместо `<select>`
- Активная кнопка подсвечивается акцентным фоном

### 1.5 Кнопка профиля
- Добавить `box-shadow` в состоянии покоя для визуального веса
- Hover-эффект: `translateY(-1px)`

### 1.6 Focus-visible
- Добавить `:focus-visible` кольца на все интерактивные элементы

## Фаза 2 — TSX: Scroll-поведение и toggle языка

**Файлы: `app/components/Navbar/Navbar.tsx`**

### 2.1 Scroll listener
```tsx
const [scrolled, setScrolled] = useState(false);
useEffect(() => {
  const onScroll = () => setScrolled(window.scrollY > 20);
  window.addEventListener('scroll', onScroll, { passive: true });
  return () => window.removeEventListener('scroll', onScroll);
}, []);
```
- Применять `className={`${styles.dotaHeader} ${scrolled ? styles.scrolled : ''}`}` 

### 2.2 Замена select на toggle
- Убрать `<select>` с `<option>` 
- Заменить на два `<button>`: RU и EN, оба с `onClick={() => setLocale(...)}`

## Фаза 3 — Мобильное меню (полная переделка)

**Файлы: `app/components/Navbar/Navbar.tsx`, `app/components/Navbar/Navbar.module.css`**

### 3.1 Unified slide-in панель
- Оба блока (`.mainMenu` + `.headerActions`) объединяются в одну панель справа (`width: 280px`, `transform: translateX(100%)`)
- Анимация: `transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1)`
- При открытии: `transform: translateX(0)`

### 3.2 Backdrop overlay
- Полупрозрачный чёрный overlay (`rgba(0,0,0,0.5)`) позади панели
- Клик по overlay закрывает меню
- `body.overflow = 'hidden'` пока меню открыто

### 3.3 CSS-бургер → крестик
- Три горизонтальные полоски через CSS (не Boxicons)
- При `open`: верхняя旋转45°, средняя исчезает, нижняя旋转-45°
- Плавная анимация через `transition`

### 3.4 Доступность
- `aria-expanded={mobileOpen}` на бургере
- `Escape` ключ закрывает меню
- `role="dialog"` на панели
